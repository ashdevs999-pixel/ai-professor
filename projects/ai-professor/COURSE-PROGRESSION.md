# AI Professor - Course Progression Structure

## Learning Philosophy

Quick Guides → Intermediate Courses → Advanced Mastery

Each path is designed to:
1. **Quick Guides (Free)**: Get users productive in 30 minutes
2. **Intermediate Courses ($14.99)**: Deepen skills with real projects
3. **Advanced Mastery ($29.99+)**: Professional-level expertise

## Auto-Scraping Sources

For detecting new AI tools:
1. **Product Hunt AI** - https://www.producthunt.com/topics/ai
2. **TechCrunch AI** - https://techcrunch.com/category/artificial-intelligence/
3. **Hacker News** - AI-related stories
4. **Reddit r/MachineLearning** - New tool announcements
5. **Twitter/X AI Community** - Trending tools

When new tools are detected:
1. Generate guide template
2. Queue for content creation
3. Notify admin for review
4. Publish after approval
